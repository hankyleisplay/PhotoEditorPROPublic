## Overview

**Photo Editor PRO** is a closed-source, commercially licensed desktop photo editing application for Windows. It offers professional-grade Lightroom-style tone controls, HSL color mixing, spline curve editing, artistic frames, a vector doodle drawing layer, and automatic updates — all in a fast, lightweight desktop application.

---

## ✨ Features

### 🎨 Image Editing

| Category | Features |
|---|---|
| **Basic Tone** | Exposure, Contrast, Temperature, Tint, Saturation |
| **Advanced Tone** | Highlights, Shadows, Whites, Blacks |
| **Presence** | Vibrance, Clarity, Dehaze |
| **Detail & Effects** | Sharpening, Noise Reduction, Vignette, Film Grain |
| **Split Toning** | Highlight Hue/Sat, Shadow Hue/Sat, Balance |
| **HSL Color Mixer** | Per-channel Hue, Saturation, Luminance (8 color ranges) |
| **Spline Curves** | RGB Master, Red, Green, Blue curve channels |
| **Crop & Rotate** | Free/ratio crop, 90° rotation, flip H/V, fine straighten |
| **Watermark** | Custom text, size, opacity, color, position |
| **Artistic Frames** | Classic White, Polaroid Retro, Midnight Slate, Neon Glow |
| **Style Presets** | 11 built-in presets with blendable intensity slider |
| **Auto Enhance** | One-click intelligent tone optimization |
| **Compare** | Before/after toggle to preview original image |

### 🖌️ Drawing Tools
- **Brush & Eraser** with configurable size (1–100 px) and opacity (1–100%)
- **Color presets** (Red, Green, Blue, Yellow, White, Black) + **custom color picker**
- Drawings scale perfectly on high-resolution photo exports
- Full **Undo / Redo** history support

### 🔄 Auto-Update
- Automatically checks for new releases on startup
- Displays version comparison and release changelog
- Downloads and installs updates with a real-time progress bar

## 🔑 License Activation

Photo Editor PRO requires a valid license key to use.

1. Launch the application — the activation screen will appear automatically
2. Copy your **Hardware ID (HWID)** displayed on screen
3. Provide the HWID to the developer to receive your license key
4. Enter the license key to activate the software

> ⚠️ License keys are non-transferable and tied to a single machine. Each license covers one device.
